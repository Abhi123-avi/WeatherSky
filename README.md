# WeatherSky

WeatherSky is a responsive weather forecast web application built using 
HTML, CSS, JavaScript, and Tailwind CSS. 

It allows users to search weather by city name or by their current location 
using the OpenWeatherMap API.


## Features

-  Search weather by city name
-  Fetch weather using current location (Geolocation API)
-  Temperature unit toggle (°C / °F) only for current day
-  5-Day weather forecast with responsive card layout
-  Recently searched cities dropdown (stored using localStorage)
-  Extreme temperature alert (above 40°C)
-  Dynamic background change based on weather condition
-  Tailwind CSS used for responsive UI and hover effects
-  Custom UI-based error handling (no JavaScript alert())


## Technologies Used

- HTML5 (Semantic Structure)
- CSS3
- Tailwind CSS (CDN)
- JavaScript (ES6)
- OpenWeatherMap API
- Geolocation API
- Local Storage


## How to Run the Project

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Enter a city name or click "Use Location".
4. Enjoy the live weather forecast.
